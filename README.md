# EcoSense Africa

Monitoring Today. Preserving Tomorrow.

## Project Overview

EcoSense Africa is a fictional non-governmental organization website designed to present a professional international environmental brand. The site communicates an environmental mission focused on smart water monitoring, climate education, renewable energy, recycling, tree planting, clean air awareness, and community development across Africa.

The final deliverable is a multi-page static website built only with HTML5 and CSS3. It is designed to feel trustworthy, modern, clean, premium, and ready for public presentation.

## Design Decisions

### Color Palette

The palette uses the required primary green `#1E8E3E`, primary blue `#2F80ED`, accent green `#27AE60`, soft background `#F8FAF8`, dark text `#222222`, and light gray `#F2F5F3`. Green communicates environmental stewardship, blue supports water and technology themes, and the soft neutral background keeps the interface calm and premium.

### Typography

Headings use Poppins for a confident modern identity. Body copy uses Inter for readability and clean professional presentation. The layout uses clear hierarchy, generous spacing, readable line heights, and strong contrast.

### Layout Strategy

The site uses a consistent container width, reusable card systems, responsive split layouts, CSS Grid, and Flexbox. Sections are separated with whitespace, soft backgrounds, gradient panels, and restrained shadows to create a polished NGO experience.

### Accessibility Considerations

The site includes descriptive image alt text, visible form labels, semantic landmarks, fieldsets and legends for forms, a skip link, ARIA labels for navigation and social links, focus states, and reduced motion support.

### Responsiveness

The design adapts at approximately 1200px, 992px, 768px, and 576px. Multi-column layouts collapse into single-column layouts on smaller screens. The navigation becomes a CSS-only hamburger menu on tablets and phones.

### UI/UX Choices

The interface avoids clutter and uses whitespace, modern cards, soft shadows, rounded corners, subtle glass effects, and measured hover transitions. The public-facing pages are written to feel like a real environmental organization rather than a classroom exercise.
